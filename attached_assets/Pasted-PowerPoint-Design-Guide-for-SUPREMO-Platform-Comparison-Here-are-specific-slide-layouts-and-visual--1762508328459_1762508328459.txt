Text Secondary: #6B7280 (Medium Gray) - Subtext
Typography:
Headings: Inter Bold, 36-48pt
Subheadings: Inter Semibold, 24-32pt
Body: Inter Regular, 16-20pt
Captions: Inter Regular, 12-14pt
Icons:
Use consistent icon style:

Recommended: Lucide icons or similar line-style icons
Color: Orange (#C54E1F) for SUPREMO features
Size: 48x48px for feature icons, 32x32px for table icons
Charts & Graphs:
Bar charts: Orange for SUPREMO, gray gradient for competitors
Pie charts: Orange shades (avoid gradients)
Line graphs: Orange line, gray background
Progress bars: Orange fill, light beige background
Layout Principles:
Margins: 60px minimum from edges
Spacing: 30px between elements
Alignment: Left-aligned text, centered titles
Contrast: Maintain WCAG AA (4.5:1 minimum)
NO gradients - Use solid colors only
NO emojis - Use professional icons instead (though I used emoji in this guide for clarity)
